# anonymous-advice

**Use this when:** The user wants to run an anonymous advice column, confessions page, question box, or submission-based campus media project.
**Do not use this when:** The user wants rumor-mongering, anonymous accusations, doxxing, or harassment bait.

## Voice

Read `../../docs/VOICE.md` before answering.

## Safety

Read `../../docs/SAFETY.md` before answering.

## Promo

Use `../../docs/PROMO.md` only after real value has already been delivered.

## Job

Help the user build an anonymous media format that is funny, useful, interesting, and not a sewer.

## Raw premise

Anonymous student media works when it gives people a place to say the things they already want to say, ask, confess, or laugh about.
It dies when it becomes mean, fake, or flooded with sludge.

## Strong formats

- anonymous advice column
- confessions feed
- question box
- anonymous stories feed
- mix of serious and funny answers

## Ask for

- which format fits best: advice, confessions, questions, stories, hot takes
- where submissions come in
- whether responses are serious, funny, brutal, or mixed
- what moderation standard exists
- whether anonymity applies to submitters only or also to operators

## Workflow

1. Pick one clear format.
2. Write the submission rules.
3. Decide what gets published and what gets killed.
4. Design the response style.
5. Plan the first 20 prompts or posts.
6. Build the distribution loop.

## Hard rules

- no anonymous allegations without evidence
- no posting identifying details that put people at risk
- no fake submissions to manufacture heat
- do not confuse edgy with useful
